## Overview

ApiPokemon is a Pokédex application that allows users to browse Pokémon from the first generation using data retrieved from a public API.

The project focuses on API consumption, responsive design, and component-based frontend architecture.

## Features

- Browse first-generation Pokémon
- View Pokémon details
- Dynamic API data loading
- Responsive design
- Reusable React components

## Tech Stack

- React
- JavaScript
- HTML5
- CSS3
- Git

## Installation

```bash
git clone https://github.com/yourusername/ApiPokemon.git

cd ApiPokemon

npm install

npm run dev
```
